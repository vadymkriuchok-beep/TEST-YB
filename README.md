# SaveNest (demo deals site)

An original, static coupon/deals aggregator built for testing. All brands, codes, and stats are fictional placeholder content. No real site was copied.

## Files
- `index.html` — page structure
- `styles.css` — design
- `data.js` — placeholder brands/deals (edit these to change content)
- `app.js` — rendering + interactions
- `netlify.toml` — Netlify config

## Deploy to Netlify

**Drag-and-drop (fastest):**
1. Go to https://app.netlify.com/drop
2. Drag this whole folder onto the page.
3. Done — you get a live test URL.

**From Git:**
1. Push this folder to a GitHub repo.
2. In Netlify: "Add new site" → "Import an existing project" → pick the repo.
3. Leave build command empty, publish directory `.`, deploy.

No build step, no dependencies.
